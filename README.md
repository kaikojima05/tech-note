# tech-note

エンジニアリングにおける技術的な知見、備忘録、および Tips を集約するためのノートです。

## ディレクトリ構成

```text
.
├── README.md
├── db/
│   └── orm/
│       └── prisma/
│           └── dmmf.md
└── language/
    └── typescript/
        └── satisfies-use-case.md
```

## 構成の概要

- **db/**: データベース設計や操作に関する知見
  - **orm/**: 各種 ORM（Object-Relational Mapping）の活用方法
    - **prisma/**: Prisma の仕組み（DMMF 等）や実装パターン
- **language/**: 各プログラミング言語の仕様やテクニック
  - **typescript/**: TypeScript の型システムや新機能の活用法
